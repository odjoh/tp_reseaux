Dans ce GitHub se trouve l’ensemble des confs et rendus du TP demandé.

Notre but à travers ce TP est, dans un premier temps, de mettre en place dans GNS3 une configuration basique de réseau avec une topologie type. Ensuite, nous avons éprouvé cette infrastructure via différents types d’attaques comme l’ARP spoofing et le DHCP spoofing.

Puis nous sommes passés à la phase de défense et de sécurisation. C’est bien d’attaquer, mais c’est encore mieux de savoir défendre afin de se prémunir des attaques.
